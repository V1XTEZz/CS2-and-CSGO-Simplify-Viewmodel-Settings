# CS2-and-CSGO-Simplify-Viewmodel-Settings

<p>Simple way to modify ur viewmodel/hands</p>
<br>
<p>Download the config and put it: <pre>...\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg</pre></p>

<h3>Check values:</h3>
<blockquote>
  <b>HInfo</b> - <code>to get MAIN current settings</code>
  <br>Add <b>V</b> to command to get current value
</blockquote>
<h3>The Settings:</h3>
<blockquote>
  <i>Hx</i> - <code>(-2 to 2.5) [+0.5 or -0.5 only] OR use Min & Max (HxMIN/MAX)</code>
  <br><i>Hy</i> - <code>(-2 to 2) [+0.5 or -0.5 only] OR use Min & Max (HyMIN/MAX)</code>
  <br><i>Hz</i> - <code>(-2 to 2) [+0.5 or -0.5 only] OR use Min & Max (HzMIN/MAX)</code>
  <br><i>Hfov</i> - <code>(54 to 68) [+2 or -2 only] OR use Min & Max</code>
  <br><i>HPreset</i> - <code>(0 / 1) choosing presets</code>
</blockquote>
